# Taro 前端运行说明

此目录现在只负责两个目标：微信小程序（`weapp`）和 H5（`h5`）。Android 已迁移到根目录 `android` 的 Flutter 工程，iOS 已迁移到根目录 `ios` 的 SwiftUI 工程。

## 环境

- Node.js 22+
- pnpm 11.13.0（建议通过 Corepack 启用）
- 微信开发者工具（仅微信小程序需要）

## 安装与检查

```bash
cd frontend
corepack enable
pnpm install
pnpm test
pnpm typecheck
```

## 开发

```bash
# 微信小程序，产物在 dist/
pnpm dev:weapp

# H5，产物在 dist/h5/
pnpm dev:h5
```

微信开发者工具导入 `frontend/project.config.json`。H5 开发服务器地址以命令行输出为准。

## 构建

```bash
pnpm build:weapp
pnpm build:h5
pnpm build:all
```

通过 `TARO_APP_API_BASE_URL` 配置后端地址；默认值是 `http://localhost:9999/api`。容器构建时可传入同名 build argument。

本目录不再包含 React Native、Expo、Metro、EAS、Android Gradle 或 iOS Xcode 构建逻辑。
