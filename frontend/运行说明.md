# 项目运行说明

## 三个问题的解决方案

### 1. 构建工具

**当前状态：** 项目使用 Webpack5 作为构建工具

**说明：**
- 项目已配置为使用 Webpack5，构建稳定可靠
- 已移除所有 Vite 相关依赖，确保项目简洁
- Webpack5 对 Taro 小程序支持完善，推荐使用

### 2. 运行到微信开发者工具

**步骤：**

1. **构建项目**
   ```bash
   npm run build:weapp
   ```
   或者使用监听模式（开发时推荐）：
   ```bash
   npm run dev:weapp
   ```

2. **打开微信开发者工具**
   - 下载并安装最新版微信开发者工具
   - 在「设置 > 安全设置」中勾选 **服务端口**

3. **导入项目**
   - 打开微信开发者工具
   - 选择「导入项目」
   - 项目目录选择：`D:\work\my-work\NewApp\frontend\dist`
   - AppID：使用项目配置中的 `wxc2c365537f482a36`（或使用测试号）
   - 点击「导入」

4. **开发模式**
   - 运行 `npm run dev:weapp` 启动监听模式
   - 在微信开发者工具中点击「编译」按钮
   - 代码修改后会自动重新编译

**注意事项：**
- `dist` 目录是构建输出目录，不要直接修改
- 源代码在 `src` 目录
- 修改代码后需要重新构建或使用 `dev:weapp` 监听模式

### 3. UI 组件库 - NutUI

**已集成：** NutUI React Taro 版本

**配置位置：**
- `src/app.tsx` - 已配置 ConfigProvider 和样式引入
- `src/pages/login/index.tsx` - 已添加 Button 组件示例

**使用示例：**

```tsx
import { Button, Cell, Toast } from '@nutui/nutui-react-taro'

// 按钮组件
<Button type="primary" block onClick={handleClick}>
  点击按钮
</Button>

// 单元格组件
<Cell title="标题" desc="描述" />

// 提示组件
Toast.show('提示信息')
```

**更多组件：**
- 访问 NutUI 官方文档：https://nutui.jd.com/taro/react/v2/zh-CN/component/button
- 支持的所有组件都可以在 Taro 项目中使用

**常用组件：**
- Button（按钮）
- Cell（单元格）
- Toast（提示）
- Dialog（对话框）
- Picker（选择器）
- Input（输入框）
- Switch（开关）
- Checkbox（复选框）
- Radio（单选框）
- 等等...

## 项目结构

```
frontend/
├── src/              # 源代码目录
│   ├── pages/        # 页面文件
│   ├── store/        # 状态管理
│   ├── utils/        # 工具函数
│   └── app.tsx       # 应用入口
├── dist/             # 构建输出目录（微信小程序）
├── config/           # Taro 配置文件
└── package.json      # 项目依赖
```

## 常用命令

```bash
# 构建微信小程序
npm run build:weapp

# 开发模式（监听文件变化）
npm run dev:weapp

# 构建 H5
npm run build:h5

# 开发 H5
npm run dev:h5
```

## 注意事项

1. **构建输出：** `dist` 目录是构建产物，不要直接修改
2. **源代码：** 所有代码修改都在 `src` 目录
3. **样式：** 使用 SCSS，支持 CSS Modules
4. **状态管理：** 使用 Zustand
5. **UI 组件：** 使用 NutUI React Taro

## 问题排查

如果遇到问题：

1. **构建失败：** 检查 Node 版本（推荐 Node 18+）
2. **微信开发者工具无法打开：** 确保选择了正确的 `dist` 目录
3. **样式不生效：** 检查 SCSS 文件路径和导入
4. **组件不显示：** 确保已正确导入 NutUI 组件和样式
